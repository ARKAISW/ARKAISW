### What I work on

I'm interested in **market microstructure**, **multi-agent systems**, and using **reinforcement learning** to model financial dynamics. Not predicting markets — understanding what emerges when you change the rules.

---

### Projects

| | |
|---|---|
| 🏦 **[Marketmind](https://github.com/ARKAISW/Marketmind)** | Agents with distinct trading charters operating inside a continuous double auction (limit order book). The system doesn't predict markets — it *is* a market. |
| 📐 **[QuantHive](https://github.com/ARKAISW/QuantHive)** | Benchmark for learned adaptive governance in multi-agent financial systems. 12 market regimes, PPO-trained governance agents, formal failure taxonomy. |
| 🎯 **[Crypto-risk-benchmark](https://github.com/ARKAISW/Crypto-risk-benchmark)** | OpenEnv-compliant RL benchmark evaluating LLM agents on professional financial risk management, position sizing, and technical analysis. |

| 🔍 **[market-structure-log](https://github.com/ARKAISW/market-structure-log)** | Market Dislocation Fingerprint Intelligence System. |
